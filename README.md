Unofficial Rust library for the Twitter API.

This library allows you to:

* get your timeline,
* update your timeline.

[Documentation](https://docs.rs/twitter-api)

## How to use?

Add this to your `Cargo.toml`:

```toml
[dependencies]
twitter-api = "0.5"
```

See [examples](./examples).
